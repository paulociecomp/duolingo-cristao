## Visão Geral do Projeto

O **Duolingo Crente** é uma aplicação web de aprendizado bíblico gamificado. O objetivo é tornar o estudo da Bíblia mais envolvente através de:

- Exercícios interativos variados
- Sistema de recompensas (XP, níveis, conquistas)
- Competição saudável (ligas semanais)
- Motivação diária (streaks, missões)

## Tecnologias Principais

```
Backend:    Ruby on Rails 8
Banco:      PostgreSQL
Frontend:   Hotwire (Turbo + Stimulus)
CSS:        Tailwind CSS 4
Auth:       Google OAuth (OmniAuth)
```
